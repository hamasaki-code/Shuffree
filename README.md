# Shuffree
すべての「決めごと」を、無料で自由に。順番決め・チーム分け・トーナメント・総当たり戦・抽選をワンクリックで生成できるランダムツール集です。Cloudflare Pagesに対応し、BuySellAds広告のみで運営できます。
## 主な機能
- **ランダムピッカー**：改行区切りで入力した名前から1つを抽選。アニメーション付きで当選者を表示します。
- **順番決め**：名簿をシャッフルして1位〜n位の順番を自動生成します。
- **チーム分け**：チーム数または人数指定でバランスの良いグループを作成。奇数人数にも対応。
- **トーナメント表**：2〜16名の勝ち上がり表をBYE処理込みで作成します。
- **総当たり戦**：参加者全員の対戦カードを一括生成。休み（BYE）も自動設定。
- **広告プレースホルダー**：BuySellAds向けの広告枠を各画面に配置済み。
## 開発環境
- Next.js 15 + TypeScript
- Tailwind CSS 4
- Zustand（ローカルストレージへの入力保持）
- Framer Motion（結果アニメーション）
## セットアップ
```bash
npm install
npm run dev
```
開発サーバーは http://localhost:3000 で起動します。
## ビルドとデプロイ
```bash
npm run build
npm run start
```
SSGされた出力を `next-on-pages` と組み合わせることで Cloudflare Pages へデプロイできます。
